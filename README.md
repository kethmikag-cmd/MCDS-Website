# Media Circle of D.S. Senanayake College
This repository contains the official website of the Media Circle of D.S. Senanayake College. The website serves as a central hub for the Media Circle, showcasing its identity, vision, and creative work while providing information about its projects, events, and initiatives. It also serves as the official digital platform for Ninnada, the annually held all-island trillingual inter-school media competition organized by the society.

The website was developed with a focus on modern design, accessibility, responsiveness, and efficient data management. Special attention has been given to performance optimization and scalability to ensure a smooth experience for participants, schools, and visitors across a wide range of devices.

## About Ninnada
Ninnada is an inter-school media competition that provides students with an opportunity to unveil their talents in media-related fields. The competition brings together young media enthusiasts from schools across the island and encourages creative excellence and the advancement of media literacy.

## Development
The website is built using modern web technologies and integrates Firebase services for registration management and data handling.

## Deployment
The project is deployed and hosted online through Netlify. (Go to https://mcdssc.netlify.app)
